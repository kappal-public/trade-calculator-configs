# Trade Calculator Community Configs

Community-maintained broker templates and lot sizes for the [Trade Calculator App](https://github.com/kappal-public/trade-calculator-app).

## Overview

This repository contains:
- **Broker Templates**: Pre-configured charge structures for various Indian stock brokers
- **Lot Sizes**: Default lot sizes for F&O segments
- **Default Template**: Baseline charge structure for the Indian market
